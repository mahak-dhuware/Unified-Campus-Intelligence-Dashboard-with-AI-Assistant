# Unified Web Dashboard

A full-stack web dashboard with an embedded AI assistant. It connects to multiple MCP servers to fetch data in real-time based on user queries.

## Features
- AI assistant for user queries  
- Modular MCP server system  
- Real-time data fetching  
- Clean and responsive UI  

## Tech Stack
- Frontend: React  
- Backend: Node.js, Express  
- AI: OpenAI / OpenRouter API  

## How it works
User asks a question → AI understands it → relevant MCP server is called → result is shown in dashboard.

## Project Goal
To build a smart, scalable, and modular dashboard that uses AI to understand user queries and fetch real-time data from independent MCP servers, avoiding traditional heavy scraping and enabling a more flexible and efficient data integration system.
